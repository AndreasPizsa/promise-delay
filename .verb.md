## API
### Promise.delay(ms[,value])

```js
Promise
  .delay(1000,'hello')
  .then(console.log);
// -> writes 'hello' after 1000ms
```

### promise.delay(ms)

```js
myAsyncFunction()
  .delay(1000)
  .then(console.log);
// -> writes the result of `myAsyncFunction` after a delay of 1000ms 
```
